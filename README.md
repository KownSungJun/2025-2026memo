파이썬 데이터 셋 확장자
파이썬 데이터셋 확장자는 .csv (쉼표로 구분된 값), .tsv (탭으로 구분된 값), .json, .xlsx (엑셀), .pkl (파이썬 pickle), 그리고 이미지 데이터의 .jpg, .png 등이 자주 사용되며, Pandas 라이브러리 (pd.read_csv(), pd.read_excel(), pd.read_pickle() 등)를 통해 쉽게 불러올 수 있습니다. 
주요 데이터셋 확장자
.csv: 가장 보편적으로 사용되며, 쉼표(,)로 데이터 필드가 구분된 텍스트 파일입니다. Pandas 라이브러리의 `pd.read_csv()` 함수로 불러옵니다.
.tsv: 탭(Tab)으로 필드가 구분된 텍스트 파일이며, CSV와 유사합니다.
.xlsx / .xls: 마이크로소프트 엑셀 파일로, Pandas에서 pd.read_excel() 함수로 읽을 수 있습니다.
.json: JavaScript Object Notation 형식으로, 구조화된 데이터를 표현하는 데 사용됩니다.
.pkl: 파이썬 객체(DataFrame, Series 등)를 그대로 저장하고 불러올 수 있는 바이너리 파일입니다 (e.g., pd.read_pickle('data.pkl')).
.npy / .npz: NumPy 배열을 저장하는 데 사용되는 포맷입니다. 
이미지/비전 데이터셋
.jpg / .jpeg: 압축된 이미지 파일입니다.
.png: 비손실 압축 이미지 파일입니다. 
사용 예시
CSV 불러오기: import pandas as pd; df = pd.read_csv('data.csv').
Excel 불러오기: import pandas as pd; df = pd.read_excel('data.xlsx').
Pickle 불러오기: import pandas as pd; df = pd.read_pickle('data.pkl').
